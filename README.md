# Next.js Firebase Authentication App

## Tech Stack
- Next.js (App Router)
- Firebase Authentication (Email/Password)
- Tailwind CSS

## Setup Instructions

1. Clone the repository
2. Run: npm install
3. Create `.env.local` using `.env.example`
4. Add your Firebase config values
5. Run: npm run dev

## Features

- User Registration
- Login with Email & Password
- Stores full name via displayName
- Protected Home Page
- Logout functionality
- Clean error handling
- Minimal Tailwind styling
